Captcha Dashboard

Configure captcha harvesters for One-Click Gmail sessions and review best-practice setup guidance before running tasks.

Quick Start

  1. Install Google Chrome. Captcha solvers use your local Chrome browser profile.
  2. Go to Captchas and click Add Captcha to create a harvester.
  3. Optional: set a proxy in Enter Proxy if you want captcha traffic on a specific IP.
  4. Click the YouTube button and sign in to a Google account (not required for Pokémon Center).
  5. After sign-in succeeds, close the YouTube window and open the pop-out arrow window before tasks start.

Buttons & Actions

ControlWhat it does
Add CaptchaCreates another captcha harvester for additional accounts or parallel tasks.
Enter ProxyOptional. If blank, harvester uses localhost IP. For reCAPTCHA targets, this field proxy is used.
Pop-out ArrowOpens the harvester window so you can manually answer challenges.
YouTubeOpens Google sign-in flow for the harvester account session.
EraserLogs out Google and clears stored harvester state.
Trash CanDeletes the harvester.
Floppy DiskSaves current harvester settings and session-related data.

Auto Click & Auto Solve

  • Auto Click: Works across all modules. Automatically clicks eligible one-click checkbox captchas.
  • Auto Solve (Serverside): Pokémon Center only. Uses remote infrastructure and is capped at 10 enabled instances at once.
  • Auto Solve (Local AI): Pokémon Center only. Uses your local machine (GPU/CPU) for solving.

Recommendation: prefer Serverside on hosted/server environments. Use Local AI on a local PC with strong hardware.

Local AI Requirements

  • Enable Local AI once; the model downloads automatically and only reinstalls for updates.
  • Before a Pokémon Center drop, confirm solver status shows Ready and run a Test solve.
  • Minimum 8GB VRAM recommended. CPU fallback works but is significantly slower.
  • Validated examples include RTX 3060, 3070, and 4060 for stable multi-harvester usage.

Proxy Behavior Notes

  • You can solve all captchas with one harvester; multiple harvesters are optional for scale.
  • If Gmail sign-in fails repeatedly, delete the harvester and create a new one.
  • For Pokémon Center-style captcha flows, task proxy settings take priority over harvester proxy.
  • For reCAPTCHA flows, the harvester proxy field is used for captcha solving.
